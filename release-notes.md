# Release Notes

## [v1.0.0] - 2026-01-05

### Added
- Initial release of `obsidian_zid_wikilink.py`.
- Support for generating Obsidian-style wikilinks with Zettelkasten IDs (ZID).
- Automatic slug generation for file names (limited to 3 words).
- Clipboard integration for easy use in text editors.
- German character replacement support.
- AHKv2 integration documentation.
- Project structure setup: `ref/`, `docs/rfcs/`.
